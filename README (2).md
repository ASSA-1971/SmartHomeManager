# smart home device manager

this is a web application for managing smart home iot devices. it lets you catalog and track different types of smart devices like lights, thermostats, cameras, and door locks throughout your home.

## features
- organize devices by category
- track device status (online/offline/maintenance)
- record device locations and installation dates
- manage device information and settings

## technology
- asp.net core mvc
- entity framework core
- sql server
- c#
